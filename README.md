
# E-commerce Data Analysis and Customer Metrics

This project provides a comprehensive analysis of an e-commerce dataset using **Python** and **SQL**. By connecting to a MySQL database, it extracts data to generate business insights regarding sales trends, customer distribution, and financial performance.

## Dataset
The analysis is performed on a multi-table e-commerce dataset including tables for customers, orders, items, payments, and sellers,.
*   **Data Source:** (https://drive.google.com/drive/folders/1POVOIs2eeBKlCCTz9HN7Jmx75HA6-lmW?usp=drive_link)

## Technologies Used
*   **Database:** MySQL (Database: `ecommerce`)
*   **Language:** Python
*   **Libraries:** `pandas`, `numpy`, `matplotlib`, `seaborn`, `mysql.connector`

## Key Analysis and Metrics

### 1. Customer & Geographical Insights
*   **Unique Locations:** Identifies all unique cities where customers are based.
*   **State Distribution:** Ranks and visualizes the count of customers by state (e.g., SP, RJ, MG) to identify high-density regions,.
*   **Order Density:** Calculates the average number of products per order grouped by city.

### 2. Sales & Revenue Analysis
*   **Category Performance:** Measures total sales and the percentage of total revenue contributed by each product category (e.g., Bed Table Bath, Health Beauty),.
*   **Monthly Trends:** Tracks order volume per month for the year 2018.
*   **Growth Metrics:** Calculates the Year-over-Year (YoY) growth rate of total sales.
*   **Cumulative Sales:** Computes cumulative sales per month across different years.

### 3. Financial & Behavioral Metrics
*   **Installment Analysis:** Calculates the percentage of orders paid using installments.
*   **Customer Lifetime Value (CLV):** 
    *   Identifies the top 3 highest-spending customers for each year.
    *   Calculates a moving average of payment values for each customer's order history.
*   **Retention Rate:** Attempts to measure the percentage of customers making repeat purchases within a six-month window.

### 4. Seller & Product Analysis
*   **Seller Revenue:** Calculates total revenue generated by each seller and ranks them accordingly.
*   **Price Correlation:** Analyzes the relationship between product price and purchase frequency.

## Visualizations Included
The project generates several visual reports, including:
*   Bar charts for customer counts by state.
*   Monthly order volume for 2018.
*   Bar charts for top-spending customers by year.
